Preprocessing Help

Parameters

  • Connected Components Size: Remove objects smaller than this volume size. Use higher values for larger noisy objects. Typical values range between 10 - 100.

Steps

  • volumeProcess: Image preprocessing alogrithm.
  • ConnCompntwFldfill: Compute Connected Component with Flood filling method.
  • AnisoDiffuse: Image smoothing algorithm - Anisotropic Diffusion
  • GradientVecField: Compute the gradient vector field of any 3D density map.
