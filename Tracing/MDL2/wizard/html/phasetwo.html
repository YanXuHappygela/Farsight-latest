Phase Two Help

Parameters

  • Morph Strength: In graph morphological method with iteration of graph erosion and dilation steps, We set the morphology strength as the number of iterations for graph erosion and dilation. Use higher values to remove longer branches. Typical values range between 30 and 70.
  • Edge Range: If the distance between 2 points is above this distance, we assume there is no edge between them in the graph structure before generating the IW-MST. Use larger values for more connected graphs. Typical values range between 1 and 15.
  • Graph Prune Size: Prune the tree structure when the trivial branches are shorter than this length. Use larger values to remove longer branches. Typical values range between 1 and 10.
  • Weight Factor: Used to emphasize the function of either model description or the data description (tradeoff between coverage and conciseness). Use higher values for more connected graphs. Typical values range between 5 and 15.

Steps

  • BackboneExtract: Extract dendrite backbone from a list of 3D points.
  • RefiningSkeleton: Form streamlines from saddle points and seed points
  • SpineExtraction: Generate graph structure from a list of 3D points.
  • Delete Intermediary Files: This button will remove the intermediary files generated during skeletonization. Your final results will be unaffected.
